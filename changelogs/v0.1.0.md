# AniMaker v0.1.0 Release Notes

Release Date: March 20, 2026

---

## Overview

AniMaker v0.1.0 是项目的初始开发版本，提供基础的动画渲染框架和视频工作站核心功能。这是一个开发预览版本，专注于框架核心功能的实现和验证。

---

## What's New

### 框架核心

**类型系统**
- 基础 TypeScript 类型定义
- 品牌类型（ObjectId, Time, Alpha）

**渲染对象**
- 抽象 `RenderObject` 基类
- `VectorObject` 支持基本形状（圆形、矩形、多边形等）
- `TextObject` 文本渲染
- `GroupObject` 对象编组

**场景管理**
- `Scene` 类用于组织和管理动画对象
- 场景调度和时间管理

**动画系统**
- 基础动画类
- 缓动函数集合（linear, smooth, easeIn/Out 等）

**事件系统**
- 类型安全的事件处理
- 事件冒泡机制

### 视频工作站

**项目结构**
- Electron + React 基础架构
- electron-vite 构建系统

**文档**
- 完整的用户文档（安装、使用手册、项目管理）
- API 文档结构
- 开发者指南

## Installation

```bash
# Clone the repository
git clone https://github.com/your-username/animaker.git
cd animaker

# Install dependencies
pnpm install

# Start development
pnpm run dev
```
