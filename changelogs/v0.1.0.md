# Kinema v0.1.0 Release Notes

Release Date: March 20, 2026

---

## Overview

Kinema v0.1.0 是项目的初始开发版本，提供高性能的 2D 动画渲染框架和视频工作站基础架构。这是一个开发预览版本，专注于框架核心功能的实现和验证。

---

## What's New

### 框架核心 (Framework Core)

**类型系统**

- 完整的 TypeScript 类型定义
- 品牌类型（ObjectId, Time, Alpha）提供编译时安全
- 不可变数据模式

**渲染对象**

- 抽象 `RenderObject` 基类与状态管理
- `VectorObject` - 支持圆形、矩形、多边形、线条、弧形、路径
- `TextObject` - 文本渲染与字体配置
- `GroupObject` - 层级对象组合

**场景管理**

- `Scene` 类用于组织和管理动画对象
- 场景调度与时间管理
- 场景快照与状态保存

**动画系统**

- 基础 `Animation` 类与 `AnimationBuilder` 流式 API
- 多种动画类型：
  - TransformAnimation - 通用变换插值
  - FadeAnimation - 淡入/淡出动画
  - RotateAnimation - 单轴与多轴旋转
  - MoveAnimation - 线性与路径移动
  - AnimationGroup - 并行、序列、延迟组合
- 20+ 缓动函数（linear, smooth, easeIn/Out, elastic, bounce）

**时间轴系统**

- Timeline 类支持播放/暂停/停止控制
- 前进/后退播放方向
- 速度控制（0.1x 到 10x）
- KeyframeTrack 支持 4 种插值类型
- TimeMarker 系统（4 种标记类型）
- TimelineController 支持撤销/重做

**事件系统**

- 类型安全的 EventEmitter
- 父链事件冒泡
- 优先级监听器执行
- 动画事件（start, update, complete, repeat, pause, cancel, error）
- 场景事件（init, objectAdded/Removed, frameRender, update, pause）
- 渲染事件（beforeRender, afterRender, error, performance, resize）
- 全局 EventBus

**工厂函数**

- 形状工厂：Circle(), Rectangle(), Polygon(), Line(), Arc(), Path()
- 动画工厂：fade(), move(), rotate(), scale(), parallel(), sequence()
- 场景工厂：createScene(), createHDScene(), animateSequentially()
- 高级 Chain API

### 渲染引擎 (Rendering Engine)

**Canvas 2D 渲染器**

- 高性能 Canvas 2D 渲染
- 批处理操作
- 形状专用渲染器（CircleRenderer, RectangleRenderer, PathRenderer）

**WebGL 支持**

- WebGL2 图形设备
- WebGPU 设备（实验性）
- 统一的图形设备接口

**渲染管线**

- 批处理（Batching）
- 视锥剔除（Culling）
- 多通道渲染
- 渲染统计（RenderStats）

**后处理效果**

- BloomEffect - 辉光效果
- BlurEffect - 模糊效果
- ChromaticAberrationEffect - 色差效果
- ColorCorrectionEffect - 色彩校正
- VignetteEffect - 晕影效果
- PostProcessing 管道

### 导出系统 (Export System)

**导出器**

- ImageExporter - PNG/JPEG/WebP 图片序列
- GifExporter - GIF 动画导出
- VideoExporter - WebM/MP4 视频导出（MediaRecorder）
- FrameEncoder - Canvas 帧编码抽象

### 视频工作站 (Video Workstation)

**架构**

- Electron + React 桌面应用框架
- electron-vite 快速构建系统
- TypeScript 严格模式

**文档体系**

- 安装指南（系统要求、安装方式、故障排除）
- 用户手册（界面概览、编辑操作、快捷键）
- 项目管理（创建、保存、备份）
- 编辑器功能（逐帧编辑、图层管理、关键帧）
- 设置配置（界面、性能、导出）

### 开发工具 (Development Tools)

**代码质量**

- ESLint + Prettier 代码规范
- TypeScript 严格模式
- 导入排序与组织

**文档生成**

- VitePress 文档站点
- TypeDoc API 文档生成
- Markdown 格式支持

**测试框架**

- Vitest 单元测试
- Playwright E2E 测试
- 测试覆盖率报告（80%+ 目标）

pnpm run dev
