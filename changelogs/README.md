# Changelogs

This directory contains detailed release notes for each version of Kinema.

## Format

Release notes follow the format: `v{major}.{minor}.{patch}.md`

## Structure

Each release note file should include:

1. **Overview** - Brief description of the release
2. **What's New** - List of new features and improvements
3. **Breaking Changes** - Any breaking changes with migration guidance
4. **Migration Guide** - How to upgrade from previous version
5. **Contributors** - Credits for contributors
6. **Known Issues** - List of known issues (if any)
7. **Next Steps** - Preview of upcoming features
