# 设置和配置

Kinema 视频工作站提供丰富的自定义选项，让您根据工作流程调整应用行为。

## 访问设置

- 点击菜单 **编辑 > 偏好设置**（或 `Ctrl+,`）
- 设置窗口分为多个类别，可在左侧导航栏切换

## 常规设置

### 界面语言

选择应用界面语言：

- 简体中文
- English
- 日本語
- 更多语言...

### 主题

选择界面外观：

- **浅色**: 明亮界面，适合日间使用
- **深色**: 深色界面，减少眼部疲劳
- **系统**: 跟随操作系统设置

### 启动行为

- **显示欢迎页**: 启动时显示最近项目
- **自动打开上次项目**: 恢复上次工作状态
- **检查更新**: 启动时检查新版本

### 自动保存

- **启用自动保存**: 定期自动保存项目
- **间隔**: 设置保存频率（1-60 分钟）
- **最大备份数**: 保留的备份版本数量（1-50）

## 项目设置

### 默认项目配置

设置新建项目的默认参数：

| 选项     | 说明         | 默认值    |
| -------- | ------------ | --------- |
| 分辨率   | 视频画布尺寸 | 1920x1080 |
| 帧率     | 每秒帧数     | 30 fps    |
| 背景颜色 | 画布背景     | #ffffff   |
| 时长     | 默认场景时长 | 10 秒     |

### 素材导入

- **自动复制到项目**: 导入时复制素材文件
- **转换为兼容格式**: 自动转换不支持的格式
- **生成代理**: 为高分辨率视频创建低分辨率代理

### 自动备份

- **保存时创建备份**: 每次手动保存时备份
- **备份位置**: 自定义备份存储路径

## 编辑器设置

### 外观

#### 缩放级别

- 预览窗口默认缩放
- 可选：25% - 400%

#### 显示选项

- 显示网格
- 显示标尺
- 显示安全区域
- 显示对象边框

#### 颜色

- 网格颜色
- 参考线颜色
- 选中对象颜色
- 背景颜色

### 性能

#### 预览质量

- **完整**: 最高质量，较慢
- **1/2**: 中等质量，平衡
- **1/4**: 低质量，快速

#### 渲染设置

- **硬件加速**: 使用 GPU 加速渲染
- **多线程渲染**: 启用多线程处理
- **缓存帧数**: 缓存的帧数量（10-1000）

### 输入

#### 鼠标

- **灵敏度**: 鼠标移动速度
- **滚轮缩放**: 使用滚轮缩放画布

#### 快捷键

- 自定义所有操作的快捷键
- 恢复默认快捷键方案

## 时间轴设置

### 显示

- **时间格式**: 时间码或帧号
- **缩放**: 时间轴默认缩放级别
- **显示波形**: 音频轨道显示波形

### 操作

- **吸附到帧**: 拖动时自动对齐到帧
- **吸附到关键帧**: 自动对齐到关键帧
- **吸附到标记**: 自动对齐到时间标记

## 导出设置

### 视频格式

#### MP4 (H.264)

- **质量**: 低/中/高/超高
- **比特率**: 自动或手动设置
- **配置**: Baseline/Main/High

#### WebM (VP9)

- **质量**: 0-100
- **比特率模式**: VBR/CBR

#### GIF

- **帧率**: 导出帧率
- **颜色**: 256 色或更多
- **优化**: 文件大小优化

### 导出范围

- **整个时间轴**: 导出完整项目
- **选定区域**: 仅导出选中部分
- **循环**: 循环播放导出的内容

### 元数据

- **包含项目信息**: 在导出文件中嵌入项目数据
- **作者信息**: 添加作者名称
- **版权声明**: 添加版权信息

## 语言和区域

### 界面语言

设置应用显示语言

### 日期和时间格式

- 短日期格式
- 长日期格式
- 时间格式（12/24小时）

### 数字格式

- 小数分隔符
- 千位分隔符

## 快捷键

### 查看快捷键

点击 **编辑 > 快捷键** 查看完整列表

### 自定义快捷键

1. 打开快捷键设置
2. 选择要修改的操作
3. 按下新的快捷键组合
4. 点击应用

### 快捷键方案

- **默认**: 应用默认快捷键
- **Adobe Premiere**: 类似 Adobe 的快捷键
- **Final Cut Pro**: 类似 Final Cut 的快捷键
- **自定义**: 您的自定义方案

## 高级设置

### 实验性功能

启用正在开发中的功能：

- 新的渲染引擎
- 实时协作
- AI 辅助编辑

> 警告：实验性功能可能不稳定

### 开发者工具

- **显示控制台**: `Ctrl+Shift+I`
- **性能监控**: 启用性能分析
- **调试模式**: 显示调试信息

### 日志

- **日志级别**: 错误/警告/信息/调试
- **日志位置**: 查看日志文件位置
- **导出日志**: 导出应用日志用于问题报告

## 重置设置

### 重置所有设置

1. 打开设置窗口
2. 点击左下角 **重置所有设置**
3. 确认重置操作

### 重置特定类别

每个设置类别页面底部有 **重置此类别** 按钮

## 导入/导出设置

### 导出设置

1. 点击设置窗口底部的 **导出设置**
2. 选择保存位置
3. 设置保存为 `.json` 文件

### 导入设置

1. 点击 **导入设置**
2. 选择之前导出的 `.json` 文件
3. 确认导入

### 同步设置

如果启用了云同步：

- 设置将自动保存到云端
- 在其他设备上登录后自动同步
- 需要网络连接

## 命令行参数

可以通过命令行参数启动应用：

```bash
# 打开特定项目
kinema.exe /path/to/project.kinema

# 无痕模式（不保存历史）
kinema.exe --incognito

# 指定配置文件
kinema.exe --config /path/to/config.json

# 调试模式
kinema.exe --debug
```
