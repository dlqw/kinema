# 国际化

AniMaker 视频工作站支持多语言界面，目前支持英语和简体中文。

## 支持的语言

| 语言 | 代码 | 状态 |
|------|------|------|
| English | `en` | 完整支持 |
| 简体中文 | `zh` | 完整支持 |

## 切换语言

### 在应用中切换

1. 点击应用右上角的语言选择器
2. 选择您想要的语言
3. 界面将立即切换，偏好设置会自动保存

### 语言偏好持久化

语言选择会自动保存到本地存储，下次启动应用时会自动应用您选择的语言。

存储位置：`localStorage['animaker-language']`

## 添加新语言

如果您想为 AniMaker 添加新语言支持：

1. **创建翻译文件**
   ```bash
   # 在 packages/workstation/src/locales/ 下创建新语言文件
   # 例如：fr.json (法语)
   ```

2. **添加翻译内容**
   参考 `en.json` 或 `zh.json` 的结构，添加所有翻译键值对

3. **注册语言**
   在 `packages/workstation/src/i18n/index.ts` 中添加：
   ```typescript
   import fr from '../locales/fr.json'

   i18n.init({
     resources: {
       en: { translation: en },
       zh: { translation: zh },
       fr: { translation: fr }  // 新语言
     },
     // ...
   })

   // 更新语言列表
   export const languages = [
     { code: 'en', name: 'English', nativeName: 'English' },
     { code: 'zh', name: 'Chinese', nativeName: '中文' },
     { code: 'fr', name: 'French', nativeName: 'Français' }
   ]
   ```

4. **提交 Pull Request**
   将您的翻译贡献到项目

## 技术实现

### 使用的库

- **i18next**: 国际化框架
- **react-i18next**: React 集成
- **zustand**: 状态管理和持久化

### 文件结构

```
packages/workstation/src/
├── i18n/
│   └── index.ts           # i18n 配置
├── locales/
│   ├── en.json            # 英文翻译
│   └── zh.json            # 中文翻译
└── stores/
    └── languageStore.ts   # 语言偏好状态管理
```

### 使用示例

在组件中使用翻译：

```tsx
import { useTranslation } from 'react-i18next'

function MyComponent() {
  const { t } = useTranslation()

  return (
    <div>
      <h1>{t('app.title')}</h1>
      <button>{t('menu.file_new')}</button>
    </div>
  )
}
```

带参数的翻译：

```tsx
// 翻译文件：{ "welcome": "Welcome, {{name}}!" }
<p>{t('welcome', { name: 'User' })}</p>
```
