# 编辑器功能

Kinema 视频工作站的编辑器提供强大的逐帧编辑和代码编辑功能，让您精确控制视频的每一帧。

## 视图模式

编辑器支持两种核心视图模式：

### 预览视图

默认视图，用于实时预览视频效果：

- 显示渲染后的最终画面
- 支持播放控制（播放/暂停/逐帧）
- 显示安全区域和参考线
- 可调整缩放比例

### 代码编辑视图

使用代码创建和修改对象：

- 基于框架的 JavaScript/TypeScript API
- 实时语法高亮和错误检查
- 代码自动完成
- 内置 API 参考

## 逐帧编辑

### 帧导航

使用时间轴底部工具栏或快捷键：

| 操作      | 快捷键  | 功能         |
| --------- | ------- | ------------ |
| 播放/暂停 | `Space` | 切换播放状态 |
| 上一帧    | `Left`  | 后退一帧     |
| 下一帧    | `Right` | 前进一帧     |
| 跳转开头  | `Home`  | 跳到第一帧   |
| 跳转结尾  | `End`   | 跳到最后一帧 |

### 帧操作

#### 选择帧

- 点击时间轴上的刻度选择特定帧
- 拖动选择多个连续帧
- `Shift+Click` 扩展选择范围

#### 复制帧

1. 选中一帧或多帧
2. 按 `Ctrl+C` 复制
3. 移动到目标位置
4. 按 `Ctrl+V` 粘贴

#### 删除帧

1. 选中要删除的帧
2. 按 `Delete` 或右键 > 删除帧
3. 后续帧会自动前移

#### 插入帧

1. 定位到插入位置
2. 右键 > 插入帧
3. 或按 `Ctrl+Shift+I`

## 对象编辑

### 选择对象

- **点击选择**: 单击对象选中
- **框选**: 拖动鼠标框选多个对象
- **全选**: `Ctrl+A`
- **取消选择**: 点击空白区域或 `Esc`

### 变换操作

选中对象后可进行以下变换：

| 操作 | 方法         | 快捷键           |
| ---- | ------------ | ---------------- |
| 移动 | 拖动对象     | -                |
| 缩放 | 拖动角点     | `Shift` 保持比例 |
| 旋转 | 拖动旋转柄   | -                |
| 倾斜 | 拖动边框中点 | -                |

### 精确控制

在属性面板中输入精确数值：

- **位置**: X/Y 像素坐标
- **大小**: 宽度/高度
- **旋转**: 角度（0-360°）
- **不透明度**: 0-100%
- **锚点**: 变换中心点

## 图层管理

### 图层顺序

图层层级决定渲染顺序：

- 顶层图层覆盖底层
- 在项目面板中调整顺序
- 或使用快捷键：
  - `Ctrl+]` 上移一层
  - `Ctrl+[` 下移一层
  - `Ctrl+Shift+]` 置于顶层
  - `Ctrl+Shift+[` 置于底层

### 图层类型

| 类型 | 图标 | 说明         |
| ---- | ---- | ------------ |
| 图像 | 🖼️   | 静态图片对象 |
| 文本 | 📝   | 文字对象     |
| 形状 | 🔶   | 矢量图形     |
| 视频 | 🎬   | 视频片段     |
| 组   | 📁   | 对象编组     |
| 相机 | 📷   | 视图控制     |

### 编组

1. 选中多个对象
2. 右键 > 编组 (`Ctrl+G`)
3. 组可作为单一对象操作
4. 双击组进入编辑模式
5. 取消编组: `Ctrl+Shift+G`

## 关键帧动画

### 添加关键帧

1. 选中对象
2. 移动时间指针到起始位置
3. 点击属性旁的 **添加关键帧** 按钮（钻石图标）
4. 移动时间指针到结束位置
5. 修改属性值，自动创建新关键帧

### 编辑关键帧

- **选择**: 点击关键帧标记
- **移动**: 拖动关键帧改变时间
- **删除**: 选中后按 `Delete`
- **复制**: `Ctrl+C` / `Ctrl+V`

### 缓动函数

为关键帧间的过渡添加缓动效果：

| 缓动        | 效果     |
| ----------- | -------- |
| Linear      | 匀速     |
| Ease In     | 渐入     |
| Ease Out    | 渐出     |
| Ease In Out | 渐入渐出 |
| Bounce      | 弹跳     |
| Elastic     | 弹性     |

## 代码编辑

### 打开代码编辑器

- 点击工具栏的 **代码** 按钮
- 或按 `Ctrl+E` 切换

### 代码结构

每个项目包含一个主代码文件：

```typescript
import { Scene, Rectangle, Circle } from 'kinema';

// 创建场景
const scene = new Scene({
  width: 1920,
  height: 1080,
  background: '#ffffff',
});

// 添加对象
const rect = new Rectangle({
  x: 100,
  y: 100,
  width: 200,
  height: 200,
  color: '#ff0000',
});

scene.add(rect);

// 添加动画
rect.animate().to({ x: 500 }, { duration: 1000, easing: 'easeInOut' }).start();

export default scene;
```

### 实时预览

- 编辑代码后自动刷新预览
- 错误信息显示在控制台
- 使用 `console.log()` 调试

### API 参考

按 `Ctrl+Space` 查看可用 API：

- 对象构造函数参数
- 方法签名和返回值
- 事件类型

## 辅助功能

### 网格和参考线

- **显示网格**: `Ctrl+'`
- **吸附到网格**: `Shift+Ctrl+'`
- **添加参考线**: 从标尺拖动
- **删除参考线**: 拖出画布

### 安全区域

- 显示标题安全区和动作安全区
- 确保内容在不同设备上可见
- 设置 > 显示 > 安全区域

### 标尺

- **显示标尺**: `Ctrl+R`
- 测量对象距离
- 拖动参考线到标尺

## 撤销与重做

- **撤销**: `Ctrl+Z`
- **重做**: `Ctrl+Shift+Z`
- 历史记录限制：50 步
- 可在设置中调整

## 查找和替换

### 查找对象

- `Ctrl+F` 打开查找面板
- 按名称、类型查找
- 支持正则表达式

### 批量修改

1. `Ctrl+H` 打开替换面板
2. 查找对象属性
3. 设置替换值
4. 批量应用修改

## 性能优化

### 降低预览质量

在编辑时降低预览质量提升性能：

- 设置 > 性能 > 预览质量
- 选择 1/2 或 1/4 分辨率

### 渲染缓存

- 自动缓存已渲染帧
- 缓存位置: 项目文件夹/cache/
- 清理缓存: 项目 > 清理缓存
