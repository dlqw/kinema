# 视频工作站文档

欢迎使用 AniMaker 视频工作站！这是一款基于 Electron 的桌面应用，专为视频编辑和动画制作设计。

## 快速开始

1. [安装指南](./installation.md) - 系统要求和安装步骤
2. [用户手册](./user-guide.md) - 基本使用方法
3. [项目管理](./projects.md) - 创建和管理项目

## 功能特性

### 核心功能

- **项目管理**: 创建、保存、管理多个视频项目
- **素材导入**: 支持图片、视频、音频等多种格式
- **时间轴编辑**: 直观的非线性编辑界面
- **逐帧编辑**: 精确控制每一帧
- **代码编辑**: 使用 JavaScript/TypeScript API 创建动画

### 高级功能

- **关键帧动画**: 专业的动画关键帧系统
- **多轨道编辑**: 视频、音频、特效独立轨道
- **图层管理**: 完整的图层顺序和编组功能
- **实时预览**: 高性能的实时渲染预览
- **多种导出**: MP4、WebM、GIF 等格式

## 技术信息

### 系统架构

视频工作站基于以下技术构建：
- **Electron**: 跨平台桌面应用框架
- **React**: 用户界面框架
- **Vite**: 快速的开发构建工具
- **AniMaker Framework**: 底层动画渲染引擎
