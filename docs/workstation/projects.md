# 项目管理功能

Kinema 视频工作站提供完整的项目管理功能，帮助您高效组织和管理多个视频项目。

## 项目文件结构

每个 Kinema 项目包含以下文件：

```
MyProject/
├── MyProject.kinema      # 项目文件（JSON格式）
├── assets/                 # 项目资源文件夹
│   ├── images/
│   ├── videos/
│   └── audio/
└── cache/                  # 渲染缓存
```

## 新建项目

### 创建新项目

1. 启动应用后，点击欢迎界面的 **"新建项目"**
2. 或点击菜单 **文件 > 新建项目** (`Ctrl+N`)
3. 配置项目参数：
   - **项目名称**: 显示在应用标题栏
   - **保存位置**: 项目文件存储路径
   - **视频设置**: 分辨率、帧率、背景色

### 项目模板

新项目可选择以下模板：

| 模板    | 分辨率    | 帧率  | 适用场景     |
| ------- | --------- | ----- | ------------ |
| HD      | 1920x1080 | 30fps | 标准高清视频 |
| Full HD | 1920x1080 | 60fps | 流畅动画     |
| 4K      | 3840x2160 | 30fps | 超高清视频   |
| Square  | 1080x1080 | 30fps | 社交媒体     |
| Mobile  | 1080x1920 | 30fps | 竖屏视频     |

## 项目设置

### 修改项目设置

点击 **文件 > 项目设置** 可修改：

#### 视频设置

- **分辨率**: 自定义宽度/高度
- **帧率**: 24/30/60 fps
- **背景颜色**: 默认画布背景
- **时长**: 默认场景时长（秒）

#### 项目信息

- **项目名称**: 修改显示名称
- **描述**: 项目备注信息
- **创建日期**: 自动记录
- **修改日期**: 自动更新

### 素材管理

#### 导入素材

支持拖拽导入或通过菜单导入：

- 图片：PNG, JPG, SVG, WEBP
- 视频：MP4, WEBM
- 音频：MP3, WAV, OGG

#### 素材库功能

项目面板中的素材库提供：

- **搜索**: 快速定位素材
- **排序**: 按名称/类型/日期排序
- **预览**: 鼠标悬停预览
- **重命名**: 右键重命名素材
- **删除**: 移除未使用的素材

#### 素材清理

点击 **项目 > 清理未使用素材** 自动移除项目中未引用的素材文件。

## 保存项目

### 自动保存

- 应用默认每 5 分钟自动保存
- 可在 **设置 > 常规** 中调整间隔

### 手动保存

- 点击 **文件 > 保存** (`Ctrl+S`)
- 或点击工具栏保存按钮

### 另存为

- 点击 **文件 > 另存为** (`Ctrl+Shift+S`)
- 创建项目副本

## 最近项目

启动应用后，欢迎界面显示最近打开的项目：

- 点击项目卡片快速打开
- 右键可：
  - 从列表中移除
  - 在资源管理器中显示
  - 显示项目信息

可设置最近项目数量（**编辑 > 偏好设置 > 常规**）。

## 项目导出

### 导出项目文件

将项目打包为 `.kinema.zip` 文件：

1. 点击 **文件 > 导出项目**
2. 选择包含素材的选项：
   - 仅项目文件
   - 包含所有素材
   - 仅包含使用的素材

### 导入项目

从 `.kinema.zip` 恢复项目：

1. 点击 **文件 > 导入项目**
2. 选择项目文件
3. 选择恢复位置

## 项目备份

### 自动备份

- 每次保存时自动创建备份
- 备份文件位于项目目录下的 `backups/` 文件夹
- 保留最近 10 个备份版本

### 手动备份

1. 点击 **文件 > 创建备份**
2. 选择备份位置
3. 备份将包含完整的项目状态

### 恢复备份

1. 点击 **文件 > 恢复备份**
2. 选择要恢复的备份版本
3. 确认恢复操作

## 项目统计

查看 **项目 > 项目信息** 了解：

- 对象数量
- 总时长
- 素材使用情况
- 文件大小
- 渲染统计

## 多项目管理

### 同时打开多个项目

- 点击 **文件 > 新窗口** 打开新项目
- 在窗口间拖放素材和对象
- 复制粘贴对象跨项目

### 项目库

在 **设置 > 项目库** 中设置默认项目文件夹，快速访问所有项目。
