## 界面概览

AniMaker 视频工作站采用经典的非线性编辑界面布局：

```
+--------------------------------------------------+
| 菜单栏: 文件 编辑 视图 帮助                        |
+--------------------------------------------------+
|          |                      |               |
|  项目    |      预览窗口          |   属性面板    |
|  面板    |                      |               |
|          |                      |               |
+--------------------------------------------------+
|                     时间轴                        |
+--------------------------------------------------+
```

### 主要区域说明

- **项目面板**（左上）：显示项目中的所有素材和场景
- **预览窗口**（中央）：实时预览视频效果
- **属性面板**（右上）：编辑选中对象的属性
- **时间轴**（底部）：排列和编辑视频轨道

### 顶部控制栏

顶部控制栏包含以下功能：

- **语言选择器**: 切换界面语言（支持英文和中文）
- **主题切换**: 在亮色和暗色主题之间切换
- **状态指示**: 显示当前应用状态（就绪、加载中等）

## 语言和主题设置

### 切换语言

1. 点击右上角的语言选择器下拉菜单
2. 选择您想要的语言：
   - **English** - 英文界面
   - **中文** - 简体中文界面
3. 界面将立即切换，偏好设置会自动保存

语言偏好会持久化保存，下次启动应用时会自动应用您选择的语言。

### 切换主题

1. 点击右上角的主题切换按钮（☀️/🌙 图标）
2. 主题会立即切换：
   - **亮色** - 适合日间使用
   - **暗色** - 适合夜间使用
   - **跟随系统** - 自动跟随操作系统的主题设置

主题设置会自动保存，下次启动应用时应用。

### 恢复默认设置

如果需要重置语言和主题设置：

1. 关闭应用
2. 清除应用数据：
   - Windows: `%APPDATA%/animaker-workstation`
   - macOS: `~/Library/Application Support/animaker-workstation`
   - Linux: `~/.config/animaker-workstation`
3. 重新启动应用

## 创建项目

### 新建项目

1. 点击菜单栏 **文件 > 新建项目**，或按 `Ctrl+N`
2. 设置项目名称和保存位置
3. 选择视频分辨率（预设：1920x1080）
4. 设置帧率（默认：30fps）
5. 点击"创建"

### 打开项目

- 点击 **文件 > 打开项目**，或按 `Ctrl+O`
- 选择 `.animaker` 项目文件

### 项目设置

点击 **文件 > 项目设置** 可修改：
- 视频分辨率
- 帧率
- 背景颜色
- 默认时长

## 项目管理

### 项目面板功能

项目面板是您管理所有视频项目的中心位置：

- **新建项目**: 创建新的空白项目
- **打开项目**: 打开已保存的项目
- **最近项目**: 快速访问最近编辑的项目
- **项目属性**: 查看和修改项目设置

### 项目属性

每个项目包含以下属性：

| 属性 | 说明 | 默认值 |
|------|------|--------|
| 项目名称 | 项目的显示名称 | My Animation |
| 保存位置 | 项目文件的存储路径 | 用户文档/AniMaker |
| 分辨率 | 视频画布尺寸 | 1920x1080 |
| 帧率 | 每秒帧数 | 30 fps |
| 背景颜色 | 默认画布背景 | #ffffff |
| 时长 | 项目总时长 | 10 秒 |

### 项目操作

#### 保存项目

定期保存您的工作：

1. 点击 **文件 > 保存项目**，或按 `Ctrl+S`
2. 项目将保存到 `.animaker` 文件
3. 所有素材和编辑状态都会被保存

#### 另存为

创建项目副本：

1. 点击 **文件 > 另存为**
2. 选择新的保存位置和名称
3. 原项目不会被修改

#### 关闭项目

1. 点击 **文件 > 关闭项目**
2. 如果有未保存的更改，系统会提示保存
3. 项目关闭后返回欢迎界面

### 项目模板

新项目可以选择以下模板：

| 模板 | 分辨率 | 帧率 | 适用场景 |
|------|--------|------|----------|
| HD | 1920x1080 | 30fps | 标准高清视频 |
| Full HD | 1920x1080 | 60fps | 流畅动画 |
| 4K | 3840x2160 | 30fps | 超高清视频 |
| Square | 1080x1080 | 30fps | 社交媒体 |
| Mobile | 1080x1920 | 30fps | 竖屏视频 |

### 自动保存

应用具有自动保存功能：

- 每 5 分钟自动保存项目
- 自动保存位置与项目文件相同
- 可以在设置中调整自动保存间隔

## 导入素材

### 支持的素材类型

| 类型 | 格式 |
|------|------|
| 图片 | PNG, JPG, WEBP, SVG |
| 视频 | MP4, WEBM, MOV |
| 音频 | MP3, WAV, OGG |

### 导入方法

1. 点击项目面板的 **"导入素材"** 按钮
2. 选择文件或拖拽文件到项目面板
3. 素材将自动添加到项目资源库

## 时间轴编辑

### 添加对象到时间轴

1. 从项目面板拖拽对象到时间轴
2. 对象将创建为新轨道
3. 调整对象的起始时间和持续时间

### 编辑对象

选中时间轴上的对象后：
- **移动**: 拖拽对象改变位置
- **调整时长**: 拖拽对象边缘
- **删除**: 按 `Delete` 键

### 关键帧动画

1. 选中对象
2. 在属性面板中点击属性旁的 **钻石图标** 添加关键帧
3. 移动时间指针到新位置
4. 修改属性值，自动创建新关键帧

### 轨道操作

- **添加轨道**: 点击时间轴左侧的 **"+"** 按钮
- **轨道类型**: 视频、音频、特效
- **调整顺序**: 上下拖拽轨道

## 预览与导出

### 预览

- **播放/暂停**: 空格键或点击播放按钮
- **跳转**: 点击时间轴或拖动时间指针
- **循环播放**: 点击循环按钮

### 导出视频

1. 点击 **文件 > 导出视频**
2. 选择导出格式：
   - MP4 (H.264)
   - WebM (VP9)
   - GIF 动图
3. 设置质量参数：
   - 分辨率
   - 帧率
   - 比特率
4. 选择保存位置
5. 点击"导出"

### 导出设置

| 格式 | 适用场景 | 特点 |
|------|----------|------|
| MP4 | 通用视频 | 兼容性好，文件较小 |
| WebM | 网络播放 | 开源格式，质量高 |
| GIF | 短动画 | 循环播放，适合演示 |

## 快捷键

### 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建项目 |
| `Ctrl+O` | 打开项目 |
| `Ctrl+S` | 保存项目 |
| `Ctrl+Shift+S` | 另存为 |

### 编辑操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+C` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+X` | 剪切 |
| `Delete` | 删除选中对象 |

### 播放控制

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `Home` | 跳转到开头 |
| `End` | 跳转到结尾 |
| `Left/Right` | 逐帧前进/后退 |
| `Ctrl+Left/Right` | 跳转到上一个/下一个关键帧 |

### 视图操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl++` | 放大 |
| `Ctrl+-` | 缩小 |
| `Ctrl+0` | 适应窗口 |
| `Ctrl+1` | 100% 视图 |
