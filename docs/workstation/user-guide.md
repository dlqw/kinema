## 界面概览

AniMaker 视频工作站采用经典的非线性编辑界面布局：

```
+--------------------------------------------------+
| 菜单栏: 文件 编辑 视图 帮助                        |
+--------------------------------------------------+
|          |                      |               |
|  项目    |      预览窗口          |   属性面板    |
|  面板    |                      |               |
|          |                      |               |
+--------------------------------------------------+
|                     时间轴                        |
+--------------------------------------------------+
```

### 主要区域说明

- **项目面板**（左上）：显示项目中的所有素材和场景
- **预览窗口**（中央）：实时预览视频效果
- **属性面板**（右上）：编辑选中对象的属性
- **时间轴**（底部）：排列和编辑视频轨道

## 创建项目

### 新建项目

1. 点击菜单栏 **文件 > 新建项目**，或按 `Ctrl+N`
2. 设置项目名称和保存位置
3. 选择视频分辨率（预设：1920x1080）
4. 设置帧率（默认：30fps）
5. 点击"创建"

### 打开项目

- 点击 **文件 > 打开项目**，或按 `Ctrl+O`
- 选择 `.animaker` 项目文件

### 项目设置

点击 **文件 > 项目设置** 可修改：
- 视频分辨率
- 帧率
- 背景颜色
- 默认时长

## 导入素材

### 支持的素材类型

| 类型 | 格式 |
|------|------|
| 图片 | PNG, JPG, WEBP, SVG |
| 视频 | MP4, WEBM, MOV |
| 音频 | MP3, WAV, OGG |

### 导入方法

1. 点击项目面板的 **"导入素材"** 按钮
2. 选择文件或拖拽文件到项目面板
3. 素材将自动添加到项目资源库

## 时间轴编辑

### 添加对象到时间轴

1. 从项目面板拖拽对象到时间轴
2. 对象将创建为新轨道
3. 调整对象的起始时间和持续时间

### 编辑对象

选中时间轴上的对象后：
- **移动**: 拖拽对象改变位置
- **调整时长**: 拖拽对象边缘
- **删除**: 按 `Delete` 键

### 关键帧动画

1. 选中对象
2. 在属性面板中点击属性旁的 **钻石图标** 添加关键帧
3. 移动时间指针到新位置
4. 修改属性值，自动创建新关键帧

### 轨道操作

- **添加轨道**: 点击时间轴左侧的 **"+"** 按钮
- **轨道类型**: 视频、音频、特效
- **调整顺序**: 上下拖拽轨道

## 预览与导出

### 预览

- **播放/暂停**: 空格键或点击播放按钮
- **跳转**: 点击时间轴或拖动时间指针
- **循环播放**: 点击循环按钮

### 导出视频

1. 点击 **文件 > 导出视频**
2. 选择导出格式：
   - MP4 (H.264)
   - WebM (VP9)
   - GIF 动图
3. 设置质量参数：
   - 分辨率
   - 帧率
   - 比特率
4. 选择保存位置
5. 点击"导出"

### 导出设置

| 格式 | 适用场景 | 特点 |
|------|----------|------|
| MP4 | 通用视频 | 兼容性好，文件较小 |
| WebM | 网络播放 | 开源格式，质量高 |
| GIF | 短动画 | 循环播放，适合演示 |

## 快捷键

### 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建项目 |
| `Ctrl+O` | 打开项目 |
| `Ctrl+S` | 保存项目 |
| `Ctrl+Shift+S` | 另存为 |

### 编辑操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+C` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+X` | 剪切 |
| `Delete` | 删除选中对象 |

### 播放控制

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `Home` | 跳转到开头 |
| `End` | 跳转到结尾 |
| `Left/Right` | 逐帧前进/后退 |
| `Ctrl+Left/Right` | 跳转到上一个/下一个关键帧 |

### 视图操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl++` | 放大 |
| `Ctrl+-` | 缩小 |
| `Ctrl+0` | 适应窗口 |
| `Ctrl+1` | 100% 视图 |
