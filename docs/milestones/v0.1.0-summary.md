## 📊 项目概览

AniMaker是一个高性能2D动画渲染框架，使用Electron开发视频工作站应用。

### 技术栈

- **核心框架**: TypeScript 5.5 + Canvas 2D
- **渲染引擎**: WebGPU/WebGL2 支持
- **桌面应用**: Electron + React 19
- **构建工具**: electron-vite
- **状态管理**: Zustand
- **国际化**: i18next
- **测试框架**: Vitest + Playwright
- **文档**: VitePress + TypeDoc

## ⚠️ 技术债务

### 高优先级

1. **测试失败**: 172个测试用例失败
   - Easing测试: 浮点精度问题 (使用toBe而非toBeCloseTo)
   - Animation测试: API不匹配 (MoveAnimation, RotateAnimation)
   - Rendering测试: RenderAPI未定义，Canvas模拟缺失
   - Integration测试: 测试设置和辅助函数缺失

2. **导出冲突修复**: 已修复多个动画类的重复导出
   - FadeAnimation, RotateAnimation, MoveAnimation, AnimationGroup

### 中优先级

3. **视频渲染未验证**: video/目录功能未测试
   - FFmpeg集成未验证
   - 场景渲染未测试

4. **类型定义不完整**: 部分类型在types/animation.ts和animation/Animation.ts重复定义

### 低优先级

5. **文档更新**: API文档需要更新以反映最新API
6. **CI/CD未配置**: 自动化测试流水线
