> 创建日期: 2026-03-19
> 状态: 已批准

---

## 1. 概述

### 1.1 项目背景

使用 AniMaker 框架自身来制作 AniMaker 的产品介绍视频，实现"自我展示"。

### 1.2 目标

- 展示 AniMaker 框架的核心能力和易用性
- 吸引普通用户了解动画创作
- 吸引开发者使用 AniMaker 进行开发
- 通过实际视频展示框架的渲染效果

### 1.3 目标受众

**主要受众**: 普通用户（想了解动画创作的人）
**次要受众**: 开发者（想使用框架的人）

### 1.4 核心信息优先级

1. **易用性** - 简洁的 API、几行代码就能做出动画
2. **功能丰富** - 各种动画效果、后处理、导出格式
3. **性能优势** - WebGPU 高性能渲染、60fps
4. **类型安全** - TypeScript 严格类型、编译时错误检查
5. **实际应用场景** - 教育视频、数据可视化、产品演示
6. **与 Manim 对比** - 突出 AniMaker 的改进和优势
